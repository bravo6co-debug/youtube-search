📝 Claude Code 요청 프롬프트
유튜브 채널 검색 웹 애플리케이션을 만들어줘. 
단일 HTML 파일, 설치 불필요.

[기본 검색 기능]
- 검색창에 키워드 입력 → 유튜브 채널 검색
- 각 결과: 채널명, 설명, 구독자 수, 썸네일, URL, 태그

[필터 1: 영상 길이]
- 쇼폼 (1분 미만)
- 롱폼 (1분 이상)
- 전체 (필터 미적용)

[필터 2: 업로드 기간]
- 1개월
- 2개월
- 6개월
- 1년
- 전체

[필터 3: 구독자 대비 조회수 비율 (5단계)]
구독자당 평균 조회수로 계산:

1단계: 구독자당 조회수 < 0.1x (저조)
2단계: 구독자당 조회수 0.1x ~ 0.5x (부진)
3단계: 구독자당 조회수 1x (기준) ← 1:1 기준
4단계: 구독자당 조회수 1x ~ 5x (우수)
5단계: 구독자당 조회수 > 5x (최우수)

[계산 로직]
- 채널의 전체 조회수 ÷ 구독자 수 = 구독자당 평균 조회수
- 이 비율을 5단계로 분류

[UI 구성]
1. 검색바 + API 키 입력
2. 3개의 필터 패널 (좌측 또는 상단)
   - 영상 길이 라디오 버튼
   - 기간 선택 드롭다운
   - 비율 단계 체크박스 (중복 선택 가능)
3. 검색 결과 카드 형태로 표시
   - 채널 정보
   - 구독자 수
   - 전체 조회수
   - 구독자당 평균 조회수 (계산 후 표시)
   - 비율 단계 배지 (예: "4단계 - 우수")

[유튜브 API 활용]
- search API: 채널 검색
- channels API: 채널 통계 (조회수, 구독자수, 동영상수)
- videos API: 동영상 정보 (길이, 업로드 날짜)
- videos.contentDetails.duration: ISO 8601 형식 파싱해서 1분 기준으로 분류

[필터 동작 방식]
- 필터 적용 후 API 재호출 (또는 결과 클라이언트 필터링)
- 여러 필터 동시 적용 가능 (AND 조건)
- 결과 개수 표시

[UX]
- 로딩 상태 표시
- 에러 처리 (API 할당량 초과 등)
- 필터 초기화 버튼
- 결과 없음 메시지
- 반응형 디자인
