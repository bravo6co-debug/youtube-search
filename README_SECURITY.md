# 🔐 API 키 보안 가이드

## ⚠️ 중요 사항

이 프로젝트는 유튜브 API 키를 사용합니다. **절대 API 키를 코드에 하드코딩하거나 공개 저장소에 올리지 마세요!**

## ✅ 안전한 사용 방법

현재 구현은 **로컬 스토리지**를 사용하여 API 키를 저장합니다:
- 브라우저에서 직접 HTML 파일을 열 때 API 키를 입력하면 자동으로 저장됩니다
- 다음에 파일을 열면 저장된 키가 자동으로 불러와집니다
- **로컬 스토리지는 해당 브라우저에만 저장되므로 안전합니다**

## 🚫 하지 말아야 할 것

1. ❌ API 키를 코드에 직접 작성
2. ❌ API 키가 포함된 파일을 Git에 커밋
3. ❌ 공개 저장소(GitHub, GitLab 등)에 API 키 노출

## 📝 API 키 발급 방법

1. [Google Cloud Console](https://console.cloud.google.com/) 접속
2. 새 프로젝트 생성 또는 기존 프로젝트 선택
3. "API 및 서비스" > "라이브러리" 이동
4. "YouTube Data API v3" 검색 후 활성화
5. "사용자 인증 정보" > "사용자 인증 정보 만들기" > "API 키" 선택
6. 생성된 API 키 복사

## 🔒 추가 보안 권장사항

- API 키 사용량 제한 설정 (Google Cloud Console에서 설정)
- 불필요한 API 권한 제거
- 정기적으로 API 키 재생성
- 키 유출 시 즉시 삭제 및 재생성
